## Step 1: Upload Files ‚úÖ
```
1. Log into InfinityFree Control Panel
2. Click "File Manager"
3. Open the "htdocs" or "public_html" folder
4. Click "Upload" button
5. Select ALL files from: c:\xampp\htdocs\hotel_system-1\
   (Tip: Select all with Ctrl+A, then ZIP them first)
6. Upload the ZIP file
7. Right-click ZIP ‚Üí Extract
8. Delete the ZIP file
```

## Step 2: Create MySQL Database ‚úÖ
```
1. Go to "MySQL Databases" in control panel
2. Click "Create"
3. Database name: hotel_appointments
4. Username: [choose a username]
5. Password: [choose a strong password]
6. Click "Create Database"
7. SAVE these credentials - you'll need them next!
```

## Step 3: Update db.php ‚úÖ
```
1. In File Manager, open: db.php
2. Replace with your credentials:

   $host = 'localhost';
   $db = 'youraccount_hotel_appointments';
   $user = 'youraccount_username';
   $pass = 'your_password_here';

3. SAVE the file
```

## Step 4: Initialize Database ‚úÖ
```
1. Open your browser
2. Go to: https://your-domain.infinityfree.app/setup_db.php
3. Wait for the page to load
4. You should see success message
5. GO BACK to File Manager
6. DELETE setup_db.php (security!)
```

## Step 6: Create Admin Account ‚úÖ
```
Option A: Register and use phpMyAdmin
- In InfinityFree, go to "MySQL Databases"
- Click "phpMyAdmin"
- Select your database
- Go to "users" table
- Edit your user row
- Set role = 'admin'
- Save

Option B: Contact InfinityFree support to do it
```

## ‚ùå TROUBLESHOOTING

### "Database connection failed"
‚Üí Check db.php credentials match exactly what InfinityFree shows

### "404 File not found"
‚Üí Make sure files are in htdocs/public_html folder

### "Can't login to admin"
‚Üí Make sure user role is set to 'admin' in database

### "Emails not sending"
‚Üí Check mail.php SMTP settings for InfinityFree
